# CompilerTests

Clone the repo:

```
git clone git@github.com/joshuajerome/CompilerTests
```

## Running Tests
1) Run CHECKSUB.sh:
   
   ```
   ./CHECKSUB.sh [path to phase3.tar] examples.tar
   ```

## Adding Test Files
Within CompilerTests folder...

1) *cd* examples
2) add .c file and output file to examples.
3) tar examples
4) push changes
